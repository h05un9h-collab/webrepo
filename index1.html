먼저 여러분의 홈페이지를 개설하신 것을 환영합니다. 이 서버 매뉴얼은 여러분의 홈페이지를 운영하기 위해 필요한 기본적인 사항을 설명하기 위해 작성되었습니다. 그외의 질문은 webmaster@newseoul.com 으로 하시거나 newseoul.com 의 메인 홈페이지를 참조 하시기 바랍니다.

  Telnet 사용법  
 

서버로의 접속
서버셋팅이 완료되었다는 메일을 받으신 후 회원께서는 Telnet 으로 접속하기위해 가입하실 당시 기재하셨던 Login 과 패스워드를 사용하실 수 있습니다. 텔넷접속을 하기 위해서는 widows의 시작 -> 실행창에서 "telnet 도메인(호스팅을 신청한 도메인)" 을 하시면 윈도우에서 제공하는 기본 텔넷 프로그램이 실행됩니다. 혹은 좀더 나은 방법은 새롬을 이용하는 방법으로 새롬의 접속 주소로 "신청하신 도메인" 을 입력하시면 됩니다.

접속 주소 : 신청하신 도메인
Login ID : E-mail로 통지
Password : 가입하신 패스워드

 

 
  텔넷에서 Email 보는 법  
  서버에는 기본 팩키지로 PINE 가 설치되어 있습니다. 서버로 접속하셔서 Email을 보시려면 pine 을 치시면 됩니다.

 

 
  초기 홈페이지 작성  
 

디렉토리를 붙이지 않은 홈페이지의 기본주소를 쳤을 때 표시되는 홈페이지로 유닉스에서는 보통 index.html 이나 index.htm 을 사용하고 있습니다. PHP를 사용하는 분은 index.php나 index.php3로 사용기도 합니다. newseoul.com 서버에서 홈페이지가 실행되는 기본 디렉토리는 접속후 ls 를 실행시켰을 때 보이는 www 디렉토리 입니다. ftp 로www 디렉토리로 들어신후 index.html을 업로드하시거나 Telnet 에서는pico나 vi 에디터로 간단한 홈페이지를 작성후 index.html로 저장시키면 됩니다.

**중요 : 초기페이지는 반드시 index.html 이어야 합니다.

  
  FTP 사용법  
  먼저 자주 가시는 세어웨어 사이트로 가시어 다른 사람들이 자주 다운받는 FTP 프로그램을 다운로드 받습니다. 예를 들어 Cuteftp 는 인기있는 프로그램중에 하나입니다. 이 프로그램을 설치 후 구동하고 접속할 주소에 신청하신 도메인, ID 와 패스워드 사용하시면 회원의 기본 디렉토리인 /home/회원디렉토리에 접속하시게 됩니다. 이때 홈페이지를 업로드 하시려면 www 디렉토리를 클릭하여 들어가신 후에 이 디렉토리에 업로드 하시면 됩니다. 혹은 기타 원하시는 파일을 서버에 두시려고 할 때 디렉토리 어디에나 새 디렉토리를 만드시어 보관하실 수 있습니다. 참고로 newseoul.com 에서는 anonymous ftp 를 허가 하지 않고 있으니 파일 다운로드 사이트를 운영하실 수 없습니다.

 

 
  Email 사용법

  CGI-BIN  
  회원 계정의 www 디렉토리 안에는 cgi-bin 이라는 디렉토리가 있습니다. 이곳에는 웹에서 실행가능한 파일 즉 웹프로그램을 설치할 수 있는 디렉토리로 유저계정에서는 유저URL/cgi-bin 이 됩니다. newseoul.com 에서는 기본적으로 게시판과 방명록을 이곳의 board 디렉토리를 설정하여 드리므로 자유로이 게시판과 방명록을 만드시어 사용하실 수 있습니다. 또한 기타의 perl 이나 CGI 프로그램 또한 이곳에 설치하시어 구동하시면 됩니다.

  유닉스 기본명령어  
 
ls 현재의 디렉토리의 내용물을 보고자 할 때 사용한다 여기서 -l 옵션은 파일의 사이즈와 퍼미션을 같이 확인하고자 할 때 사용하며 -a 는 숨겨진 파일들을 확인할 때 사용한다. 두 옵션을 같이 사용하고자 할 때 ls -al 등으로 사용할 수도 있다.
pwd 사용자의 현재의 디렉토리 표시
cd 디렉토리 변경하고자 할 때 사용
mkdir 새디렉토리를 만들고자 할 때 사용(예:mkdir xxx)
rm 파일이나 디렉토리를 지우고자 할 때 사용 (예: rm xxx) 여기서 -f 옵션은 강제로 지우고자 할 때 사용하고 이 옵션을 사용하지 않을시에는 지워도 되냐는 메시지가 나온다. 또한 디렉토리를 지우고자 할때는 -r 을 사용한다
cp 파일을 카피하고자 할 때 사용한다. (예 : cp 카피할파일명 새파일명)
mv 파일이나 디렉토리를 다른곳이나 다른이름으로 옮길 때 사용한다.(예: mv 옮길파일 옮겨갈파일)
find 필요한 파일을 찾고자 할 때 사용한다.
Mail 메일을 보내고자 할 때 사용한다.
Passwd 패스워드를 변경할 때 사용한다.
